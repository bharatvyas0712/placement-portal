# 🎓 Placement Portal System

A Full Stack Placement Management System developed using **React.js, Node.js, Express.js, and MySQL**.
This application helps colleges manage students, companies, job postings, and placement applications efficiently.

---

## 🚀 Features

### 👨‍💼 Admin Panel

* Admin Login (JWT Authentication)
* Dashboard Overview Statistics
* Manage Students
* Manage Companies
* Manage Job Postings
* View Applications
* Delete Students & Jobs

### 🎓 Student Module

* Student Registration & Login
* Upload Resume
* View Available Jobs
* Apply for Jobs
* Track Application Status

### 🏢 Company Module

* Company Registration & Login
* Post Job Openings
* View Applicants
* Manage Job Listings

---
